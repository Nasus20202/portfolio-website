# Portfolio Website (v2)

This is a statically generated personal portfolio built with [Astro](https://astro.build/), [React](https://reactjs.org/), and [Tailwind CSS](https://tailwindcss.com/). The theme mimics a "Git/Terminal" developer environment.

## Quick Start

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm dev
```
